Como Usar o Norma Executor

1. Obtenha sua Key

Abra o terminal do Windows (CMD) e digite:

-ri key

Isso irá mostrar a key temporária que você deve usar para acessar o executor.

2. Inicie o Norma Executor

Dê duplo clique no arquivo injector.py ou abra pelo terminal com:

python injector.py

3. Insira a Key

Quando solicitado, cole a key copiada do CMD. O executor irá validar e iniciar.

4. Escreva ou carregue scripts

Use o editor interno para criar scripts ou clique em "Abrir" para importar um arquivo `.lua`.

5. Injete no Roblox

Com o Roblox aberto em segundo plano, clique em "Injetar".

WARNING - Run the '-ri key' command before opening de Injector.py using 'python injector.py'
